# Руководство пользователя

## Регистрация

1. Перейдите на страницу регистрации.
2. Введите имя пользователя.
3. Введите пароль и подтвердите его.
4. Нажмите кнопку **«Зарегистрироваться»**.

## Авторизация

1. Откройте страницу входа.
2. Введите имя пользователя и пароль.
3. Нажмите кнопку **«Войти»**.

## Каталог книг

На главной странице доступны:

- просмотр списка книг;
- поиск книг по названию;
- фильтрация по категориям;
- просмотр подробной информации о книге.

## Корзина

Для добавления книги нажмите кнопку **«Добавить в корзину»**.

В корзине можно:

- увеличить количество экземпляров;
- уменьшить количество экземпляров;
- удалить книгу из корзины;
- просмотреть общую стоимость заказа.

## Оформление заказа

1. Перейдите в корзину.
2. Проверьте список выбранных книг.
3. Нажмите кнопку **«Оформить заказ»**.

После оформления:

- создаётся новый заказ;
- книги сохраняются в составе заказа;
- корзина автоматически очищается.

## Просмотр заказов

В личном кабинете доступны:

- список оформленных заказов;
- подробная информация по каждому заказу;
- текущий статус заказа.

## Административная панель

Администратор может:

- добавлять, изменять и удалять книги;
- управлять авторами;
- управлять категориями;
- просматривать и изменять заказы;
- управлять пользователями.
