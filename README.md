# Система управления книжным интернет-магазином

Веб-приложение, разработанное на языке Python с использованием фреймворка Django, предназначенное для управления книжным интернет-магазином. Система предоставляет каталог книг, регистрацию пользователей, корзину покупок, оформление заказов, а также административную панель для управления данными.

## Технологии

- Python 3.12
- Django
- SQLite
- Bootstrap
- OpenPyXL
- Pillow

## Функционал

- Каталог книг
  - просмотр списка книг;
  - поиск книг по названию;
  - фильтрация по категориям;
  - просмотр подробной информации о книге.
- Пользователи
  - регистрация;
  - авторизация;
  - выход из системы;
  - профиль пользователя.
- Корзина
  - добавление книг в корзину;
  - изменение количества товаров;
  - удаление товаров;
  - подсчёт общей стоимости заказа.
- Заказы
  - оформление заказа из корзины;
  - автоматическое создание позиций заказа;
  - просмотр списка заказов;
  - просмотр деталей заказа.
- Административная панель, в которой реализовано управление книгами, авторами, категориями, заказами, пользователями, экспорт/импорт каталога книжной продукции в формате Excel.

## Тестирование

Проект покрыт автоматическими тестами для:
- каталога книг;
- пользователей;
- корзины;
- заказов;
- импорта и экспорта книг.

Для запуска тестов:
`python manage.py test`
