# Система управления книжным интернет-магазином

Веб-приложение для управления каталогом книг, пользователями, корзиной и заказами.

## Технологии

- Python 3.12
- Django
- SQLite
- Bootstrap

## Функционал

- Каталог книг
- Авторизация пользователей
- Корзина
- Заказы
- Административная панель
